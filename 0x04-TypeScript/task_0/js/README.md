## Task 0: Student Table with TypeScript
**Project Description** 

This project demonstrates:

- Creating a TypeScript interface Student.

- Populating an array of students (studentsList) with multiple student objects.

- Dynamically generating an HTML table using Vanilla JavaScript to display student information.

Features

- Interface Student with firstName, lastName, age, and location.

- studentsList array containing at least 2 student objects.

- Dynamic HTML table creation using:

document.createElement

appendChild

Table headers and rows automatically generated from the studentsList.
